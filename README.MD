### База данных
Создана база данных из трёх сущностей
1. User
    + id (int:11)
    + login (varchar:12)
    + password 
    + role можно выбрать только из трёх вариатов:
      + 'Пользователь',
      + 'Администратор',
      + 'Специалист'
2. Task
3. Category

### Код в соотвесвии со структурой
#### Технологии
- **Python** - основной язык программирования
- **Peewee** - ORM для подключения к БД
- **MySQL** - ситема управления базами данных
- **PyMySQL** - библиотека для подключения к MySQL
- **bcrypt** - библиотека для хешироваания паролей

#### Модели
##### User (Пользователи)
 - `id` - первичный ключ
 - `login` - логин уникальный, максимум 12 символов
 - `password` - хешированный пароль
 - `role` - роль пользователя: 
   - 'Пользователь',
   - 'Администратор',
   - 'Специалист'
 - `is_active` - при создании True, Альтернатива удаления
 - `fullname` - полное имя пользователя, максимум 150 символов

#### Контроллеры
За работу с пользователями отвечает `Controllers/UserController.py`
```python
# регистрация пользователя
from Controllers.UserController import UserController
UserController.registration(
        login='user',
        password='user'
    )
# вывод списка пользователей из БД
for row in UserController.get():
    print(row.id, row.login, row.password, row.role, row.is_active, row.fullname)

# аутентификация пользователя
UserController.auth('user','user')

```
